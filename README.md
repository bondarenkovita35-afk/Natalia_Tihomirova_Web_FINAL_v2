# Natalia Tihomirova — Web CV + Personal Letter (SV/EN)

## How to use
1) Replace images:
- `assets/photo.jpg`  (Natalia photo)
- `assets/logo.png`   (VS LiteCode Studio logo)

2) Open `index.html` in browser, or use VS Code extension **Live Server**.

## Pages
- `index.html`  — CV
- `letter.html` — Personal letter (separate page)

## Language
Use the **SV / EN** toggle in the header.
Swedish text is extracted from the provided PDFs (unchanged).  
English version is a professional summary (can be replaced with full translation if needed).

## PDFs
- CV download button downloads `Natalia_Tihomirova_CV_Final_Full_SE.pdf`
- Letter download button downloads `Natalia_Tihomirova_Personligt_Brev.pdf`
